Introdução ao HTML

Voltar

Entendi que o HTML é a linguagem de marcação usada para estruturar páginas da web. Ele funciona com elementos representados por tags, e cada parte do conteúdo da página (como textos, links e imagens) é organizada através dessas tags.



Trabalhando com Formulários

Voltar

Aprendi que os formulários permitem coletar dados dos usuários. Vimos elementos como <input>, <textarea>, <select> e botões, que servem para criar campos de texto, caixas de seleção, listas suspensas e envio de informações.



Estrutura e Formatação

Voltar

Nessa parte, entendi como organizar melhor o conteúdo de uma página, usando títulos <h1> até <h6>, parágrafos <p>, destaques como negrito, itálico, sublinhado e até elementos como marcação de texto.



Trabalhando com Mídias

Voltar

Aqui aprendi a inserir imagens com <img>, adicionar áudios com <audio> e vídeos com <video>. Também vimos atributos como controls e autoplay para controlar como esses elementos são exibidos.



Criando Tabelas

Voltar

Aprendi a montar tabelas usando <table>, organizando linhas com <tr> e colunas com <td> e <th>. As tabelas são muito úteis para apresentar dados de forma organizada.



Entendendo HTML Semântico

Voltar

Nessa parte vi a importância de usar tags semânticas, como <header>, <main>, <footer>, <article> e <section>. Isso ajuda os navegadores e buscadores a entenderem melhor o conteúdo da página, além de melhorar a acessibilidade.
